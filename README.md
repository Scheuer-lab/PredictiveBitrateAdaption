# webrtc-server setup
(based on https://github.com/borjanebbal/webrtc-node-app)

0) Install node js

1) Install dependencies for webrtc server when in webRTC folder: Run terminal: npm install express socket.io

Install mkcert and create certificate for server:

3) Create mkcert certificate while in the webRTC folder via terminal: mkcert <host ip-address>

4) run Server through terminal: node .\server.js

5) Connect to server: Enter the following URL into your browswer: https://"host-ip-address":3000 

6) On the Browser UI,  select the same room for both clients to connect them
